# Tipos de Imóveis Suportados

Os tipo de imóveis suportados hoje pelo portal Chaves na Mão são definidos de acordo com a finalidade de cada imóvel (Residencial/Comercial/Outros). Os nomes devem estar exatamente iguais aos tipos suportados pelo portal.

Verifique a lista de tipos para cada finalidade:

- [Residencial](/arquivo/dados-suportados/residencial)
- [Comercial](/arquivo/dados-suportados/comercial)

## Relação com Finalidade

### Residencial (RE)
Tipos mais comuns:
- Apartamento
- Casa
- Casa de Condomínio
- Sobrado
- Kitnet
- Cobertura

### Comercial (CO)
Tipos mais comuns:
- Sala Comercial
- Loja
- Galpão
- Prédio Comercial
- Ponto Comercial

::: tip Dica
Alguns tipos podem servir tanto para uso residencial quanto comercial. Neste caso, use a tag `<finalidade>` para especificar o uso atual.
:::
