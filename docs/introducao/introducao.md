# Introdução

Os dados serão enviados ao portal Chaves na Mão via link para o arquivo XML. O cliente deverá disponibilizar uma URL na internet para que seja efetuada a leitura e/ou download deste arquivo.

## Como Funciona

O arquivo deverá ficar disponível para acesso a qualquer momento, onde o portal poderá baixar as informações e atualizar os dados no sistema do portal.

De preferência, o XML gerado pelo sistema do cliente deverá ser dinâmico, sendo atualizado automaticamente com os imóveis contido no sistema do cliente. 

## Requisitos Técnicos

- Arquivo XML disponível via URL pública
- Codificação UTF-8 obrigatória
- Estrutura de tags case sensitive
- Atualização dinâmica recomendada
